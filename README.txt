=== Pomodoro App ===

Kullanım:
1. PomodoroApp.exe'ye çift tıklayın
2. Tarayıcınızda uygulama otomatik açılacaktır
3. Kapatmak için görev çubuğundaki ikona sağ tıklayıp "Çıkış" seçeneğini seçin

Not: İlk açılışta Windows Güvenlik Uyarısı alabilirsiniz. 
"Daha fazla bilgi" -> "Yine de çalıştır" seçeneğini kullanabilirsiniz.
